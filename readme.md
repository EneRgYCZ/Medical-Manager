# Medical Manager

## 1.Install dependencies

#### A. Navigate to `/server` directory and execute:
```
-npm install
```

#### B. Navigate to the main directory and execute:
```
-npm install
```

## 2.Run `ngrok`
```
-ngrok http "your port or domain"
```

## 3.Run expo cli

#### Navigate to the main directory and execute:
```
-npm start
```
